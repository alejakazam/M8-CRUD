-> Sistema de Gestión de Tickets (CRUD con Django) <-

Aplicación web desarrollada por Alejandra Duarte para Talento Digital, 
Proyecto realizado con Django que permite gestionar tickets de soporte para la resolución de problemas. Este sistema implementa operaciones CRUD completas (Crear, Leer, Actualizar y Eliminar).

I. Características
- Crear nuevos tickets de soporte
- Listar todos los tickets registrados
- Editar información de tickets existentes
- Eliminar tickets
- Interfaz sencilla utilizando Bootstrap

II. Tecnologías utilizadas
- Python 
- Django 
- HTML5 
- Bootstrap 5

III. Instalación
1.Clona el repositorio:
git clone https://github.com/tu-usuario/tu-repo.git
2.cd tu-repo
3.Crea un entorno virtual:
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows
4.Instala dependencias:
pip install -r requirements.txt
5.Aplica migraciones:
python manage.py migrate
6.Ejecuta el servidor:
python manage.py runserver
