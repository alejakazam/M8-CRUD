# Sistema de Gestión de Tickets (CRUD con Django)

Aplicación web desarrollada por Alejandra Duarte para Talento Digital.

Proyecto realizado con Django que permite gestionar tickets de soporte para la resolución de problemas.
El sistema implementa operaciones CRUD completas (Crear, Leer, Actualizar y Eliminar).

---

## I. Características

* Crear nuevos tickets de soporte
* Listar todos los tickets registrados
* Editar información de tickets existentes
* Eliminar tickets
* Interfaz sencilla utilizando Bootstrap

---

## II. Tecnologías utilizadas

* Python
* Django
* HTML5
* Bootstrap 5

---

## III. Instalación

### a. Clonar el repositorio

```bash id="1e0yqf"
git clone https://github.com/tu-usuario/tu-repo.git
cd tu-repo
```

### b. Crear entorno virtual

```bash id="y2u8kn"
python -m venv venv
```

Activar entorno:

```bash id="y7t9xp"
# Linux / Mac
source venv/bin/activate

# Windows
venv\Scripts\activate
```

### c. Instalar dependencias

```bash id="b3k5ds"
pip install -r requirements.txt
```

### d. Aplicar migraciones

```bash id="n6r2vz"
python manage.py migrate
```

### e. Ejecutar el servidor

```bash id="z4m8qw"
python manage.py runserver
```
